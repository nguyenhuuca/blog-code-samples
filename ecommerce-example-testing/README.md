# Testing Example: eCommerce

A sample project to demonstrate various kinds of tests.

## Run Tests

```sh
# Run all tests 
./gradlew check

# Run only unit + component tests
./gradlew test

# Run only integration tests
./gradlew integrationTest

# Run only end-to-end tests
./gradlew end2endTest

# Run only UI tests
./gradlew uiTest
```
